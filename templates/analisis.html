Descripción de tablas
Hechos
fact_siniestros

Tabla central del modelo. Cada fila representa un siniestro vial registrado en Bogotá. Contiene las métricas del evento (hora, cantidad de vehículos y actores) y las llaves foráneas que la conectan con todas las dimensiones.

{{ conteos['fact_siniestros'] | int | format_miles }} registros — fuente: SINIESTROS (Open Data Bogotá)
Dimensión temporal
dim_fecha

Desglosa cada fecha en atributos jerárquicos: año, mes, día, trimestre, nombre del mes y día de la semana. Permite análisis temporales sin cálculos en la consulta.

{{ conteos['dim_fecha'] | int | format_miles }} fechas únicas
Dimensión geográfica
dim_localidad

Las 20 localidades de Bogotá D.C., cada una con su código oficial y nombre. Compatible con capas SIG del Distrito para análisis geoespacial.

{{ conteos['dim_localidad'] }} localidades
Dimensión de severidad
dim_gravedad

Tres niveles de gravedad ordenados: Con Muertos (1), Con Heridos (2) y Solo Daños (3). El campo nivel permite ordenar y filtrar por criticidad.

{{ conteos['dim_gravedad'] }} niveles
Dimensión de tipo
dim_clase

Categoría del siniestro: Choque, Atropello, Caída de ocupante, Volcamiento, Incendio, entre otros. Derivada del campo CLASE de la fuente original.

{{ conteos['dim_clase'] }} clases
Tabla puente M:N
bridge_siniestro_hipotesis

Resuelve la relación muchos-a-muchos entre siniestros e hipótesis de causa. Un mismo evento puede tener múltiples causas registradas.

Tabla puente M:N
bridge_siniestro_vehiculo

Registra cada vehículo involucrado en un siniestro con su tipo (automóvil, moto, bus…). Un siniestro puede involucrar múltiples vehículos.

Cada actor vial involucrado: conductor, motociclista, peatón, ciclista o pasajero. Incluye su estado (ileso, herido, muerto) y sexo.

Preguntas que responde el modelo
¿Cuáles localidades tienen más siniestros con muertos?
SELECT l.nombre, COUNT(*) AS total
FROM fact_siniestros s
JOIN dim_localidad l USING (localidad_id)
JOIN dim_gravedad  g USING (gravedad_id)
WHERE g.nivel = 1  -- Con Muertos
GROUP BY l.nombre
ORDER BY total DESC LIMIT 5;
¿En qué franja horaria ocurren más siniestros?
SELECT hora,
       COUNT(*) AS total,
       ROUND(COUNT(*) * 100.0 / SUM(COUNT(*)) OVER (), 1) AS pct
FROM fact_siniestros
WHERE hora IS NOT NULL
GROUP BY hora
ORDER BY total DESC LIMIT 5;
¿Cuál es la hipótesis de causa más frecuente por año?
SELECT f.anio, h.descripcion, COUNT(*) AS total
FROM bridge_siniestro_hipotesis bh
JOIN fact_siniestros s  USING (siniestro_id)
JOIN dim_hipotesis   h  USING (hipotesis_id)
JOIN dim_fecha       f  USING (fecha_id)
GROUP BY f.anio, h.descripcion
ORDER BY f.anio, total DESC;
¿Qué tipo de actor vial tiene más muertes?
SELECT ca.descripcion, COUNT(*) AS muertos
FROM bridge_siniestro_actor ba
JOIN dim_condicion_actor ca USING (condicion_id)
WHERE ba.estado = 'MUERTO'
GROUP BY ca.descripcion
ORDER BY muertos DESC;
